Personalized Desktop Setup

Almost certainly not a programming project, but I'm still proud of how nice my desktop looks now. I played with Windows settings and registry editing a bunch (i.e. no icons on desktop, Windows bar hides unless scrolled across) to get it feeling pretty clean, and then used a software called Rainmeter to get it past the finish line. Now, instead of clicking on shortcuts, I just press the Windows key, type the name of the application into the search bar, and press enter. It's pretty sweet. All of my backgrounds are storyboard panels from Kiki's Delivery Service.

Key Achievements:

  • Desktop is always permanently organized
  • No more clutter
  • Widgets add useful information while not being distracting
  • Computer feels a lot nicer to use now

Technologies: Rainmeter Windows OS
